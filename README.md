# Shiny Icons

## Install
### For the server
Head to server directory and install the packages
````bash
cd ./server/
npm install
`````
### For the client
Head to client directory and install the packages
````bash
cd ./client/
npm install
````

## Run
#### **Install steps are required the first time**
### Start the server
````bash
cd ./server/
npm start
````
### Start the client
````bash
cd ./client/
npm start
````
